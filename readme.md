## webassembly-examples

From Simple To Complex. A complete collection of [webassembly](http://webassembly.org/) examples.

## Usage

For 1 - 8:

```sh
# start
node ./#-path-to-example/index.js

# rebuild
cd #-path-to-example && wast2js script-name.wat -o script-name.js
```

For 9+ see readme.
